## User Story

Como **SMB owner**, quero escolher em quais plataformas publicar cada conteúdo, para ter controle sobre onde cada post aparece (ex: vídeo só no Instagram e TikTok, artigo só no LinkedIn).

---

## Acceptance Criteria

**AC1 — Opção de escolher plataformas na confirmação**

```
Dado que Carlos está na etapa de confirmação antes de publicar
Quando o bot exibe o resumo
Então inclui opção: "Publicar em todas ou escolher? (todas/escolher)"
```

**AC2 — Seleção individual de plataformas**

```
Dado que Carlos digitou "escolher"
Quando o bot exibe as plataformas conectadas
Então Carlos vê:

"Selecione as plataformas (ex: 1 3 para Instagram e LinkedIn):
1. Instagram ✅ conectado
2. TikTok ✅ conectado
3. LinkedIn ✅ conectado
4. YouTube ✅ conectado"

Quando Carlos digita "1 2"
Então apenas Instagram e TikTok são selecionados para publicação
```

**AC3 — Mínimo de uma plataforma**

```
Dado que Carlos não selecionou nenhuma plataforma
Quando tenta confirmar
Então bot responde: "Selecione ao menos uma plataforma."
```
