## User Story

Como **gestora de agência**, quero selecionar qual cliente estou gerenciando ao iniciar o bot, para publicar conteúdo nas contas certas sem misturar dados entre clientes.

---

## Acceptance Criteria

**AC1 — Detecção de conta tipo agência após identificação**

```
Dado que Marina se identificou com seu CPF
Quando o sistema detecta que seu plano é 'agency' (ou tem clients cadastrados)
Então o bot NÃO exibe o menu principal imediatamente
E exibe: "Olá Marina! Selecione o cliente que vamos trabalhar agora:

1. Clínica Beleza Total (@clinica_beleza)
2. Restaurante Sabor & Arte (@sabor_arte)
3. Studio Fitness Pro (@studiofitness)

Digite o número do cliente:"
```

**AC2 — Seleção válida de cliente**

```
Dado que Marina está vendo a lista de clientes
Quando Marina digita "2"
Então o bot registra client_id na sessão ativa
E exibe: "✅ Trabalhando com: Restaurante Sabor & Arte
Plataformas conectadas: Instagram, Facebook, TikTok

O que vamos fazer hoje?"
E exibe o menu principal
```

**AC3 — Troca de cliente durante a sessão**

```
Dado que Marina está com contexto do cliente A ativo
Quando Marina digita "trocar cliente" ou "/cliente"
Então o bot limpa o client_id da sessão
E exibe novamente a lista de clientes para seleção
```

**AC4 — Agência sem clientes cadastrados**

```
Dado que Marina se identificou mas não tem clientes cadastrados
Quando o sistema verifica clients da agência
Então o bot exibe: "Você ainda não tem clientes cadastrados. Acesse postai.app/agency para adicionar seus primeiros clientes."
```

**AC5 — Publicação sempre no contexto do cliente selecionado**

```
Dado que Marina selecionou o cliente "Restaurante Sabor & Arte"
Quando Marina completa um fluxo de publicação
Então o sistema usa os api_tokens do restaurante (não os de Marina)
E registra a campanha vinculada ao client_id do restaurante
```

---

## Notas Técnicas

- Novo schema: `agency_clients` (agency_user_id, client_name, client_instagram_handle, etc.)
- `whatsapp_sessions` precisa de campo `active_client_id` (nullable)
- Para contas não-agência, `active_client_id = null` e o sistema usa os tokens do próprio usuário
- Lista de clientes limitada a 10 por página (paginação via WhatsApp para agências com muitos clientes)
