## User Story

Como **criador de conteúdo**, quero enviar copy + vídeo pelo WhatsApp e publicar automaticamente em todas as minhas plataformas conectadas, para eliminar o trabalho manual de postar em cada rede separadamente.

---

## Acceptance Criteria

**AC1 — Coleta de copy**

```
Dado que o usuário selecionou "1 - Publicar conteúdo" no menu
Quando o estado muda para 'waiting_for_copy'
Então o bot responde: "📝 Qual é a legenda do seu post? (você pode usar emojis e hashtags)"
```

**AC2 — Coleta de vídeo**

```
Dado que o usuário enviou a copy
Quando o bot recebe mensagem de texto em estado 'waiting_for_copy'
Então o bot salva a copy em campaign_draft.copy
E muda estado para 'waiting_for_video'
E responde: "✅ Copy salva! Agora envie o vídeo (MP4 ou MOV, máx 500MB)"
```

**AC3 — Upload de vídeo**

```
Dado que o usuário está no estado 'waiting_for_video'
Quando o usuário envia um arquivo de mídia
Então o backend baixa o arquivo usando o token da WhatsApp API
E faz upload para S3 com key: media/{user_id}/{timestamp}-{message_id}
E salva a URL em campaign_draft.video_url
E muda estado para 'waiting_for_thumbnail'
E responde: "🎬 Vídeo recebido! Envie a capa (JPG/PNG) ou digite 'pular'"
```

**AC4 — Confirmação antes de publicar**

```
Dado que o usuário completou copy + vídeo (thumbnail opcional)
Quando o usuário envia thumbnail ou "pular"
Então o bot exibe resumo para confirmação:

"📋 Revisão antes de publicar:

📝 Copy: {primeiros 100 chars}...
🎬 Vídeo: ✅
🖼️ Capa: ✅ / ⏭️ Sem capa

📱 Publicar em:
• Instagram ✅
• TikTok ✅  
• LinkedIn ✅
• YouTube ✅

Digite 'confirmar' para publicar ou 'cancelar'"
```

**AC5 — Publicação paralela em todas as plataformas**

```
Dado que o usuário digitou 'confirmar'
Quando o backend inicia a publicação
Então usa Promise.allSettled para publicar em todas as plataformas simultaneamente
E o tempo total de publicação é < 5 segundos para 4 plataformas
E muda estado para 'publishing'
```

**AC6 — Feedback de resultado por plataforma**

```
Dado que as publicações foram processadas (com sucesso ou falha)
Quando todas as promises são resolvidas
Então o bot envia:

"✅ Publicado com sucesso!

• Instagram: ✅ ver post → instagram.com/p/{id}
• TikTok: ✅ ver post → tiktok.com/@user/video/{id}
• LinkedIn: ✅ ver post
• YouTube: ⚠️ Falhou (token expirado) → reconectar: postai.app/settings"

E estado retorna para 'menu'
```

**AC7 — Retry automático em falhas**

```
Dado que uma plataforma retornou erro temporário (5xx, timeout)
Quando o backend detecta a falha
Então tenta novamente até 3x com backoff exponencial (1s, 2s, 4s)
Se ainda falhar após 3 tentativas: registra como failed e notifica usuário
```

---

## Notas Técnicas

- Implementar com `Promise.allSettled` — nunca usar `Promise.all` (uma falha não deve bloquear as demais)
- Vídeo no WhatsApp tem TTL — baixar para S3 imediatamente ao receber
- Tamanhos máximos por plataforma: Instagram Reels 100MB, TikTok 4GB, LinkedIn 200MB, YouTube 256GB
- Comprimir vídeo se necessário antes de enviar para cada plataforma
- Campanhas salvas em `campaigns` com status: `draft`, `publishing`, `published`, `partial_failure`, `failed`
