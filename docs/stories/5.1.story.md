## User Story

Como **usuário**, quero ver o histórico dos meus últimos posts pelo WhatsApp e pelo dashboard web, para acompanhar o que foi publicado, quando e com qual status.

---

## Acceptance Criteria

**AC1 — Histórico via WhatsApp (últimos 5)**

```
Dado que o usuário está identificado
Quando digita "3" ou "histórico" no menu
Então o bot responde com os 5 posts mais recentes:

"📋 Seus últimos posts:

1. 02/04 - Instagram, TikTok ✅ | LinkedIn ❌
   'Novo vídeo sobre...'

2. 31/03 - Todas ✅
   'Dica do dia...'
..."
```

**AC2 — Histórico por cliente (agência)**

```
Dado que Marina tem o cliente "Restaurante Sabor & Arte" selecionado
Quando digita "histórico"
Então o bot exibe os últimos 5 posts DESSE cliente específico
E exibe: "Histórico de: Restaurante Sabor & Arte"
```

**AC3 — Dashboard web (últimos 30 posts)**

```
Dado que o usuário está em /dashboard
Quando a página carrega
Então exibe tabela com: data, copy (truncada), plataformas, status, links
E filtros por: plataforma, status (publicado/falhou), período
```

**AC4 — Detalhes de uma publicação**

```
Dado que o usuário clica em uma publicação no dashboard
Quando a página de detalhes abre
Então exibe: copy completa, thumbnail, resultado por plataforma, links dos posts, timestamp de cada publicação
```
