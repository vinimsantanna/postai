## User Story

Como **usuário cadastrado**, quero conectar minhas redes sociais (Instagram, TikTok, LinkedIn, YouTube) via OAuth no dashboard, para que o bot possa publicar nas minhas contas automaticamente.

---

## Acceptance Criteria

**AC1 — Lista de plataformas disponíveis**

```
Dado que o usuário está em /settings/social
Quando a página carrega
Então exibe as 4 plataformas com status: Conectado (@handle) ou "Conectar" (botão)
```

**AC2 — Fluxo OAuth para cada plataforma**

```
Dado que o usuário clica em "Conectar Instagram"
Quando é redirecionado para OAuth do Meta
E autoriza o acesso
Então o callback salva access_token e refresh_token em api_tokens (criptografado)
E exibe o Instagram como "Conectado: @seu_perfil"
```

**AC3 — Token expirado detectado durante publicação**

```
Dado que o token de uma plataforma expirou
Quando o publisher tenta publicar e recebe 401
Então o sistema tenta refresh automático do token (se refresh_token disponível)
Se refresh falhar: notifica o usuário via WhatsApp:
"⚠️ Seu Instagram desconectou. Reconecte em: postai.app/settings/social"
```

**AC4 — Desconectar plataforma**

```
Dado que o usuário quer remover uma integração
Quando clica em "Desconectar" na plataforma
Então o sistema revoga o token na API da plataforma (se suportado)
E remove api_tokens do banco
E exibe status "Não conectado"
```

---

## Notas Técnicas

- Tokens armazenados criptografados com AES-256 via AWS Secrets Manager ou campo encrypted no Prisma
- Escopos necessários por plataforma:
  - Instagram: `instagram_content_publish`, `instagram_manage_comments`
  - TikTok: `video.upload`, `user.info.basic`
  - LinkedIn: `w_member_social`, `r_liteprofile`
  - YouTube: `https://www.googleapis.com/auth/youtube.upload`
