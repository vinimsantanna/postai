## User Story

Como **novo usuário**, quero me cadastrar no PostAI pela web informando meus dados e escolhendo um plano, para receber acesso ao bot de publicação via WhatsApp.

---

## Acceptance Criteria

**AC1 — Formulário de cadastro**

```
Dado que o visitante acessa postai.app/cadastro
Quando a página carrega
Então exibe formulário com: Nome completo, Email, CPF, Senha, seleção de plano
```

**AC2 — Validação de CPF único**

```
Dado que o usuário preenche o formulário
Quando envia o cadastro com CPF já cadastrado
Então exibe: "Este CPF já está cadastrado. Acesse sua conta."
E NÃO cria duplicata
```

**AC3 — Cadastro bem-sucedido**

```
Dado que todos os campos são válidos e CPF é único
Quando o usuário submete o formulário
Então o sistema cria o usuário no banco com status 'pending_payment'
E redireciona para checkout do Stripe com plano selecionado
```

**AC4 — Instrução pós-pagamento**

```
Dado que o pagamento foi confirmado pelo Stripe
Quando o webhook do Stripe notifica
Então o status do usuário muda para 'active'
E o sistema envia email: "Conta ativa! Salve este número no WhatsApp: +55 73 98854-8309 como 'PostAI' e envie seu CPF para começar."
```

---

## Notas Técnicas

- CPF deve ser validado no frontend E no backend
- Senha: mínimo 8 chars, hash com bcrypt (salt rounds: 12)
- LGPD: exibir checkbox de aceite dos termos + política de privacidade
