## Princípios de conversational design

1. Uma coisa por mensagem
2. Linguagem natural e friendly
3. Feedback claro de progresso
4. Erros são oportunidade de ajudar
5. Respeitar o tempo do usuário

## UX goals

- Primeiro post em <3 minutos
- Zero learning curve
- 0 momentos de confusão
- Experiência delightful

## Output esperado

- Conversation flow diagrams (todos os estados)
- Copy framework (mensagens, erros, confirmações)
- Design system (se dashboard)
- Figma prototype para user testing
